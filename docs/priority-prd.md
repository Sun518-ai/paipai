# PRD - 任务优先级功能

## 1. 概述与目标

为 paipai TodoMVC 添加任务优先级功能（P0/P1/P2/P3），让用户能够对任务进行紧急程度分类，并支持按优先级排序，紧急任务醒目显示。

## 2. 用户故事

- **作为** 派派（用户），**我希望** 给每个任务设置 P0~P3 优先级，**以便** 快速识别和处理最紧急的事项
- **作为** 派派，**我希望** 任务按优先级自动排序，**以便** 最重要的事情始终排在最前面
- **作为** 派派，**我希望** 优先级有明显的视觉区分（颜色 + 图标），**以便** 一眼就能分辨紧急程度

## 3. 功能详述

### 3.1 优先级定义

| 优先级 | 名称 | 颜色 | 图标 | 说明 |
|--------|------|------|------|------|
| P0 | 紧急 | 红色 `#EF4444` | 🔴 | 最紧急，需要立刻处理 |
| P1 | 高 | 橙色 `#F97316` | 🟠 | 重要，近期需要完成 |
| P2 | 中 | 蓝色 `#3B82F6` | 🔵 | 普通优先级 |
| P3 | 低 | 灰色 `#9CA3AF` | ⚪ | 可以稍后处理 |

- 默认优先级为 P3（低）
- 每个任务都可以单独设置优先级
- 支持随时修改优先级

### 3.2 优先级设置入口

- 在任务行中显示当前优先级的图标/颜色
- 点击优先级图标打开优先级选择菜单（Popover/Dropdown）
- 选中后自动保存到 localStorage

### 3.3 排序规则

任务列表按以下规则排序（综合排序）：

1. **置顶（pinned）任务优先**
2. **同层级按优先级排序**：P0 > P1 > P2 > P3
3. **同优先级按创建时间排序**：新创建的在前

### 3.4 优先级可视化

- 任务行的左侧，checkbox 旁边显示优先级图标
- P0 任务文字加粗
- P0 任务行左侧有一条红色边线（border-left accent）

### 3.5 新增任务时的默认优先级

- 新增任务默认优先级为 P3（最低）
- 用户可以在添加后修改

## 4. UI/UX 设计

### 4.1 优先级选择器

- 点击优先级图标弹出小型下拉菜单
- 菜单内显示 4 个优先级选项（P0~P3），带图标和名称
- 当前选中的优先级有选中态（背景色）
- 点击外部自动关闭

### 4.2 任务行布局

```
[P0图标] [checkbox] 任务文字                    [📌] [🗑️]
```

## 5. 数据模型

```typescript
interface Todo {
  id: string;
  text: string;
  done: boolean;
  createdAt: number;
  pinned: boolean;
  priority: 'P0' | 'P1' | 'P2' | 'P3'; // 新增字段，默认 'P3'
}
```

## 6. 国际化

所有新增的优先级相关文案（菜单项、提示文字）均支持中英文。
