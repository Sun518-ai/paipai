# PRD - 重复任务功能 (Recurring Tasks)

## 1. Concept & Vision

**派派的重复任务管理** — 帮助养成好习惯的任务系统。用户可以创建每日、每周、每月重复的任务，系统自动在设定时间生成新任务，完成后自动创建下一个周期的任务。

核心理念：**简单、可靠、可视化** — 让习惯养成变得轻松透明。

---

## 2. 用户故事

### 作为用户，我希望...
- 创建任务时选择重复频率（不重复/每日/每周/每月）
- 查看每个任务的重复规则
- 完成任务后，系统自动生成下一个周期的任务
- 在列表中一眼识别哪些任务是重复任务

---

## 3. 功能详述

### 3.1 重复规则设置
- **不重复（默认）**: 一次性任务，完成后不生成新任务
- **每日重复**: 每天固定时间出现
- **每周重复**: 每周特定星期几出现
- **每月重复**: 每月特定日期出现

### 3.2 任务数据模型扩展
```typescript
interface Todo {
  id: string;
  text: string;
  done: boolean;
  createdAt: number;
  pinned: boolean;
  dueDate: number | null;  // existing field
  // 新增字段
  recurring?: RecurringRule;
  completedAt?: number;
}

interface RecurringRule {
  type: 'none' | 'daily' | 'weekly' | 'monthly';
  dayOfWeek?: number;   // 0=周日, 1=周一, ..., 6=周六 (weekly)
  dayOfMonth?: number;  // 1-31 (monthly)
  lastGeneratedAt: number; // timestamp of last generation
  seriesCreatedAt: number; // original creation time
}
```

### 3.3 自动生成逻辑
- **任务完成时**: 检查 `recurring` 字段，如果存在则生成下一个周期的任务
- **页面加载时**: 检查所有重复任务，找出超过一个周期未完成的任务并提示用户

### 3.4 重复任务标记
- 重复任务在列表中显示 🔁 图标
- 悬停显示具体重复规则（如"每天"、"每周一"、"每月15号"）
- 完成的重复任务显示 ✨ 图标（表示新周期已生成）

### 3.5 编辑重复规则
- 点击任务进入编辑模式
- 可以修改/取消重复规则

---

## 4. UI 交互

### 4.1 添加任务时的重复选择
- 在输入框下方添加「重复」下拉选择器
- 默认选择「不重复」
- 选择「每周」时显示星期选择器
- 选择「每月」时显示日期选择器（1-31）

### 4.2 任务列表中的可视化
- 重复任务显示 🔁 前缀
- 悬停气泡显示完整规则描述
- 完成的重复任务显示 ✨ 图标（表示新周期已生成）

### 4.3 状态流转
```
新建任务（重复） → 进行中 → 完成 → 自动生成下一个周期任务 → 进行中 → ...
```

---

## 5. 边缘情况

| 场景 | 处理方式 |
|------|----------|
| 每月31号，但某月没有31号 | 使用该月最后一天 |
| 每周重复的任务 | 按 `dayOfWeek` 固定星期 |
| 删除重复任务 | 只删除当前实例，不影响系列 |
| 编辑重复任务 | 修改当前任务 + 更新系列规则 |
