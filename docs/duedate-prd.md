# 📅 到期提醒功能 PRD

## 1. Concept & Vision

为派派的 TodoMVC 添加截止日期管理能力，让用户不再错过任何重要任务。功能简洁直观，视觉提示醒目但不打扰——临近到期时温柔提醒，过期时明确警示。

## 2. 功能需求

### 2.1 截止日期设定
- 每个任务在创建后可设定截止日期（date picker）
- 支持清除已设定的截止日期
- 日期以友好的方式显示（如"今天"、"明天"、"3月15日"）

### 2.2 临近提醒（1天内）
- 截止日期在 24 小时内的未完成任务显示为 **橙色/amber** 提示
- 左侧边框高亮 + 背景色
- 图标使用 📅 并配合颜色变化

### 2.3 过期任务特殊显示（红色）
- 已过期的未完成任务显示为 **红色** 背景/边框
- 图标使用 ⚠️ 明确警示
- 过期任务排在最前

### 2.4 按截止日期排序
- 支持按截止日期升序排序（最近到期优先）
- 未设定日期的任务排在最后
- 置顶任务仍优先于普通任务
- 提供排序切换按钮

### 2.5 过滤器扩展
- 新增「即将到期」过滤器（dueSoon）
- 显示临近到期（≤24h）的任务

## 3. 数据模型

```typescript
interface Todo {
  id: string;
  text: string;
  done: boolean;
  createdAt: number;
  pinned: boolean;
  dueDate: number | null; // Unix timestamp (ms), null 表示无截止日期
}
```
