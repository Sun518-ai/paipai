# TodoChat 富文本卡片展示 PRD

## 1. 背景

### 1.1 项目概述
「派派点子站」是一个为派派（孙识宇小朋友）打造的创意工具集合，其中的 TodoChat 模块提供了 AI 智能对话功能。

### 1.2 当前问题
当前 TodoChat 的 AI 回复仅以纯文本形式展示，当 AI 生成任务列表、摘要、链接等结构化内容时，用户体验较差：
- 任务列表无法直接勾选完成
- 长文本没有折叠/展开功能
- 链接和数据的可读性差

### 1.3 为什么需要富文本卡片
- **更好的可读性**：结构化展示，内容分块清晰
- **交互性**：任务可直接勾选，数据可展开查看
- **美观性**：精美的卡片设计提升用户体验

## 2. 需求

### 2.1 功能需求

#### 2.1.1 卡片类型
| 卡片类型 | 标识 | 触发条件 | 交互 |
|----------|------|----------|------|
| 📝 任务卡片 | `task` | AI 回复包含 `- [ ] ` 或 `* [ ] ` 格式的任务列表 | 可勾选完成 |
| 📖 摘要卡片 | `summary` | AI 回复包含 `:::summary` 标记的区块 | 可展开/折叠 |
| 🔗 链接卡片 | `link` | AI 回复包含 URL | 显示链接预览 |
| 📊 数据卡片 | `data` | AI 回复包含 `:::data` 标记的统计数据 | 指标展示 |

#### 2.1.2 卡片格式规范
AI 回复中使用以下 markdown 标记触发卡片渲染：

```
# 任务卡片
:::task
- [ ] 任务1
- [ ] 任务2
- [x] 已完成任务
:::

# 摘要卡片
:::summary
这是长文本内容，可以折叠展开查看完整信息...
:::

# 数据卡片
:::data
{"指标1": 100, "指标2": 200}
:::

# 链接卡片
:::link
https://example.com
标题描述
:::
```

#### 2.1.3 交互要求
- 点击卡片标题/图标展开/折叠
- 任务卡片点击复选框更新状态
- 卡片状态保存在组件内部（不持久化）
- 符合项目暗黑模式

### 2.2 非功能需求

#### 2.2.1 性能
- 卡片渲染不应影响首屏加载时间
- 大量消息时考虑虚拟滚动

#### 2.2.2 兼容性
- 支持现代浏览器
- 移动端适配（响应式布局）

## 3. UI/UX 设计

### 3.1 卡片样式

#### 任务卡片
```
┌─────────────────────────────────────┐
│ 📝 任务                    [−]     │
├─────────────────────────────────────┤
│ ☐ 任务1                             │
│ ☐ 任务2                             │
│ ☑ 已完成任务                         │
└─────────────────────────────────────┘
```

#### 摘要卡片
```
┌─────────────────────────────────────┐
│ 📖 摘要                    [+]     │
├─────────────────────────────────────┤
│ 这是长文本内容，可以折叠展开...       │
└─────────────────────────────────────┘
```

#### 链接卡片
```
┌─────────────────────────────────────┐
│ 🔗 链接标题                          │
│ https://example.com                 │
│ 描述文本                            │
└─────────────────────────────────────┘
```

#### 数据卡片
```
┌─────────────────────────────────────┐
│ 📊 数据指标                          │
├─────────────────────────────────────┤
│ 指标1: 100    指标2: 200            │
└─────────────────────────────────────┘
```

### 3.2 样式规范

| 元素 | 样式 |
|------|------|
| 卡片容器 | 圆角 12px，边框，深色模式适配 |
| 卡片标题栏 | 浅色背景，图标+标题+折叠按钮 |
| 折叠状态 | 展开时 [−]，折叠时 [+] |
| 任务项 | 复选框 + 文字，完成时划线 |
