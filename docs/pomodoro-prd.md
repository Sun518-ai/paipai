# Pomodoro Timer PRD

## 背景 / 目标

在「派派点子站」中集成一个内置的番茄钟（Pomodoro Timer）功能，帮助用户专注工作/学习。派派是一个帮助用户管理想法和创意的工具，番茄钟是学习和工作中最经典的时间管理工具之一，集成此功能可以提升点子站的实用性和趣味性。

**目标：** 为用户提供一个简洁美观的内置番茄钟，支持 25 分钟专注 + 5 分钟休息的节奏，搭配视觉化圆形进度条和完成次数统计。

---

## 功能列表

### 核心功能

1. **Pomodoro 计时器**
   - 工作阶段：25 分钟（1500 秒）
   - 休息阶段：5 分钟（300 秒）
   - 支持开始、暂停、重置操作
   - 计时结束触发浏览器通知提示

2. **视觉化进度显示**
   - 圆形 SVG 进度条，实时反映剩余时间比例
   - 数字时钟格式显示剩余时间（MM:SS）
   - 状态文字指示当前阶段（专注中 / 休息中）

3. **完成次数统计**
   - 记录今日完成的番茄数量
   - localStorage 持久化存储
   - 页面刷新后数据不丢失

4. **阶段切换**
   - 工作结束自动进入休息
   - 休息结束可开始新的番茄
   - 休息阶段用不同颜色区分

### 交互设计

- **开始按钮**：绿色，开始或继续计时
- **暂停按钮**：黄色，暂停当前计时
- **重置按钮**：红色/灰色，重置当前阶段
- **完成通知**：浏览器 Notification API + 视觉提示

---

## 验收标准

1. ✅ 点击开始，计时器从 25:00 开始倒计时，每秒更新一次
2. ✅ 点击暂停，计时器停止计时
3. ✅ 点击重置，当前阶段回到初始时间
4. ✅ 25 分钟结束后，弹出浏览器通知"专注时间结束，开始休息吧！"
5. ✅ 5 分钟休息结束后，弹出通知"休息结束，开始新番茄！"
6. ✅ 圆形进度条实时反映剩余时间比例（满圆→逐渐减少）
7. ✅ 完成一个番茄后，完成次数 +1
8. ✅ 刷新页面后，完成次数和当前状态保持
9. ✅ UI 风格与项目现有页面（countdown、todomcv）保持一致
10. ✅ `npm run build` 成功通过，无 TypeScript 错误
