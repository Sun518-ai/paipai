# 🎨 HTML预览生成器 - 参数控件生成（子需求2）

## 概述

根据解析出的变量自动生成对应类型控件，支持一键生成JSON入参声明并复制。

## 功能需求

### 核心功能

1. **变量类型控件生成**
   - 根据变量名智能推断类型
   - 支持类型: text / number / password / email / url / boolean / select
   - 每个控件可独立配置类型、标签、默认值

2. **select 类型选项管理**
   - 支持自定义选项列表
   - 可动态添加/删除选项
   - 选项可编辑

3. **JSON 入参声明**
   - 一键生成 JSON 格式的参数声明
   - 一键复制到剪贴板
   - 格式: `{变量名: {type, label, default, options?}}`

4. **实时预览**
   - 右栏展示当前参数结构预览
   - 展开/收起切换
   - 复制成功视觉反馈

## 界面设计

### 布局
- 参数控件生成区位于左侧功能描述下方
- JSON 预览组件独立展示

### 组件

1. **ParamGenerator.tsx** - 参数控件生成器
   - 变量统计信息（数量/重复检测）
   - 控件列表（每个变量一个卡片）
   - 控件卡片包含：
     - 变量名标签 (code 样式)
     - 类型选择下拉框
     - 标签输入框
     - 默认值输入（根据类型不同展示不同控件）
     - select 类型的选项编辑区

2. **ParamPreview.tsx** - JSON 预览组件
   - 头部：类型统计 + 展开/收起按钮 + 复制按钮
   - 内容：JSON 格式化输出
   - 底部：参数数量统计

## 交互设计

### 控件交互
- 切换类型时自动重置默认值
- boolean 类型切换使用按钮而非 checkbox
- select 类型显示选项编辑区
- 添加选项按钮动态追加新选项

### 复制交互
- 点击复制按钮
- 成功后按钮文字变为 "✅ 已复制"
- 2秒后恢复原状态
