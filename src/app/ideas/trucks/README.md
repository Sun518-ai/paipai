# 派派工程车王国 🏗️🚜

## 功能介绍

**派派工程车王国**是一个专为3岁小朋友设计的工程车探索乐园！

### 核心功能

1. **🚜 探索模式** — 浏览10种超酷工程车，每辆都有专属emoji和有趣小知识
2. **📢 嘀嘀喇叭** — 点一点车车卡片就能听车车叫声！
3. **⭐ 收藏最爱** — 派派可以把最喜欢的车车收藏起来
4. **👀 真实见过** — 记录派派在生活中真正见过的工程车，见一次点一次！
5. **📸 照片上传** — 给车车拍照片存进百科

### 内置10种工程车

🚜 挖掘机 | 🚚 自卸卡车 | 🏗️ 吊车 | 🚛 搅拌车 | 🚜 推土机
🔵 压路机 | 🚒 消防车 | 🚜 拖拉机 | 🔧 叉车 | 🛻 平板运输车

## 飞书多维表格配置（可选）

当前使用内置默认数据，无需配置即可使用。
如需迁移到飞书云端，步骤如下：

### 1. 创建多维表格

在飞书创建一个新多维表格，记录 App Token

### 2. 设置环境变量

在部署平台添加：
```
FEISHU_TRUCK_APP_TOKEN=你的AppToken
FEISHU_TRUCK_TABLE_ID=表格ID
```

### 3. 表格字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| name | 文本 | 车辆名称 |
| category | 文本 | 类别 |
| emoji | 文本 | emoji图标 |
| funFact | 文本 | 有趣小知识 |
| color | 文本 | 主题色（如 #f59e0b）|
| bgFrom | 文本 | 背景渐变色起点 |
| bgTo | 文本 | 背景渐变色终点 |
| photoToken | 文本 | 飞书图片 file_token |
| seenCount | 数字 | 见过几次 |
| isFavorite | 复选框 | 是否最爱 |
| soundUrl | 文本 | 声音URL |

## 技术实现

- **前端**: Next.js App Router, React, Tailwind CSS
- **后端**: Next.js API Route + 飞书开放平台
- **数据持久化**: 飞书多维表格（支持本地 fallback）
- **状态管理**: localStorage + 云端同步
